# 260630_HPP_정산 흐름 비교

**현행 구조 (브랜드본부 경유)**
- 섹타나인 ↔ 브랜드본부 간 정산
- 적립분/사용분: 인보이스 처리 (VAT 없음)
- 수수료 (적립 0.4% + 사용 5%): 세금계산서 처리 (VAT 별도)
- 대금 지급: `{적립분 + 수수료(VAT 별도)} - 사용대금` → 통합 지급 (대부분 적립분이 크므로 **섹타나인이 수취**)


**직정산 구조 추가 시 **** 
- **무상 HP → 기존대로 섹타나인 ↔ 브랜드본부 정산 유지** 
- **유상 HP (HPP) → 섹타나인 ↔ 가맹점주 직정산 신설 (거래대금과 사용수수료만 직정산으로 진행= 정산 시 양수:섹타나인 수취, 음수: 브랜드 수취 case가 발생하지 않음)**
